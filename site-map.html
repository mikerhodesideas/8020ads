Game State (localStorage)
State Type Set By Used By
type PlayerType | null /setup (avatar selection) Gates /play, /play/[id], /world, /victory. Determines demo content.
world WorldId | null /setup (path choice) or /world Gates /play, /play/[id], /victory. Determines visual skin.
completed Set<number> /play/[id] (markComplete) Level map states, level unlocking, world unlocking, victory gate.
skills Set<string> Auto on demo complete (L2) Skill inventory panel in navbar.
unlockedWorlds Set<string> Auto on level complete /world page gating. L1 done = zelda. L2 = red-alert. L3 = all.
completed-proofs Set<string> / (proof landing), /proof/[type] Home page auto-redirect. 3 proofs: inbox, website, data.
badges Record<string, number> Auto-earned on conditions Victory screen. 8 badges: first-skill, skill-collector, speed-demon, explorer, level-up, full-clear, transformer, replay-master.
choiceScores Record<number, number> /play/[id] (player choices) Star ratings (1-3 per demo). Victory screen total.
